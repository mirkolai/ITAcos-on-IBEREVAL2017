You need to:

1. download data from http://stel.ub.edu/Stance-IberEval2017/data.html

2. Read the dataset/README file

3. Create a database on mysql and compile config.py file

4. You need to run, in the order indicated below, the following files:

- 000. Import database schema.py
- 001. Import training.py
- 002. Import test.py


5. After that you'll can run test and training prediction for stance (A) and gender (B).

6. Training results will be available on generated csv files. Test prediction submitted runs will be available on mysql.
